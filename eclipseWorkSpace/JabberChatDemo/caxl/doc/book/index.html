The CAXL Book

ver. CAXL-release-2014.04.10787

This book provides a friendly introduction to the Cisco AJAX XMPP Library (CAXL). CAXL gives you the tools to add presence and real-time messaging to your web applications through JavaScript libraries and user interface components that leverage the XMPP-based features of Cisco WebEx and Cisco Unified Presence.

Table of Contents

  1. Introduction: quick start, about you, CAXL overview, installing, deploying, debugging
  2. Basic Concepts: asynchronous operations, eventing, entities and entity sets, controllers
  3. Login and Logout: connect() and disconnect()
  4. Contacts and Presence: roster contacts, quick contacts, presence notifications, presence subscriptions
  5. Direct Chat: exchanging messages, chat sessions, chat states
