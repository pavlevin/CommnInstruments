removeEntity(entity)

Removes the given entity. This method is not called directly; instead call jabberwerx.Entity#remove.

This implementation uses the following algorithm:

  1. If the room is currently active and the client is currently connected, exit the room (via jabberwerx.MUCRoom#exit.
  2. Once the room is not active and/or the client is not connected, it is destroyed (via jabberwerx.Entity#destroy); this will ultimately trigger an "entityDestroyed" event for {entity}.

{jabberwerx.MUCRoom} entity
The entity to remove

if {entity} is not an instance of jabberwerx.MUCRoom
{jabberwerx.MUCRoom} room(jid)

Retreives or creates a MUCRoom for the given JID. This method implements the following algorithm:

  1. If a MUCRoom already exists in the owning client for {jid}, that instance is returned.
  2. A new MUCRoom is created for {jid}.
  3. If there is an entity for {jid} but it is not a MUCRoom, the data from the original entity is applied to the new MUCRoom (via jabberwerx.Entity#apply), and the original entity is removed from the owning client.
  4. The new MUCRoom is registered with the owning client (triggering an "entityCreated" event) and returned.
Parameters:
{jabberwerx.JID|String} jid
The room JID
Throws:

if {jid} is not an instanceof jabberwerx.JID, and cannot be converted to one.
