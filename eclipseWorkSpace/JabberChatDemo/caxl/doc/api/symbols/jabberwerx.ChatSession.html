Class jabberwerx.ChatSession


Extends jabberwerx.JWModel.

Represents a chat session with another user. There is only one ChatSession object per bare jid. The locked resource (if the session is currently locked) is specified by the jid property.

This class provides the following events:

The jid with which this chat session is communicating. If the chat session has locked onto a resource, the resource value can be found by using jabberwerx.JID#getResource.
{String} localState
Our current chat state (XEP-0085) in this chat session. It can be one of 'active', 'composing', 'paused', 'inactive' or 'gone'. State is set to active automatically when #sendMessage is called and set to gone automatically when #destroy is called.

Note: Do not set this property directly; Use #setChatState instead.
